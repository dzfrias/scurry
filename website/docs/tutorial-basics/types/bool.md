# Bool

The boolean implementation in Scurry.

## Behavior
- Supports the `!` operator
  ```
  >> !True;
  False
  ```
- Can be used with `&&` and `||`
  ```
  >> True && False;
  False

  >> True || False;
  True
  ```

## Methods
The builtin `Bool` type has no methods.
